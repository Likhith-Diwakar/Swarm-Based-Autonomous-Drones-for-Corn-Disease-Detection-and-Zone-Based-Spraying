# Swarm-Based-Autonomous-Drones-for-Corn-Disease-Detection-and-Zone-Based-Spraying
A swarm-based autonomous drone solution that uses YOLOv8-based AI detection, GPS-mapped crop analysis, DBSCAN zone clustering, and MAVLink-controlled drones to detect Northern Corn Leaf Blight and perform targeted, zone-based fungicide spraying, reducing chemical usage and operational costs.
